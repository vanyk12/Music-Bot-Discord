Создание музыкального Discord-бота — отличный проект, который поможет разобраться в программировании и работе с API. Самый популярный и понятный язык для этой задачи — Python.

Важное предупреждение: YouTube активно борется со скачиванием и трансляцией музыки через сторонние приложения, нарушающие их Условия использования (из-за этого закрылись крупные боты вроде Groovy и Rythm). Для личного сервера создать такого бота можно, но будьте готовы к тому, что YouTube иногда меняет алгоритмы, и библиотеку для парсинга придется регулярно обновлять.

Ниже приведена пошаговая инструкция, как создать базового бота с помощью библиотеки discord.py и yt-dlp.

Шаг 1: Подготовка на стороне Discord
Перейдите на Discord Developer Portal.

Нажмите New Application, введите имя бота и согласитесь с правилами.

Перейдите во вкладку Bot слева.

В разделе Privileged Gateway Intents обязательно включите Message Content Intent (чтобы бот мог читать команды в чате). Сохраните изменения.

Нажмите Reset Token и скопируйте полученный токен. Никому его не показывайте!

Чтобы добавить бота на сервер, перейдите во вкладку OAuth2 -> URL Generator.

Поставьте галочку на bot, а в появившемся блоке разрешений выберите Administrator (или настройте права вручную: чтение/отправка сообщений, подключение к голосу и разговор).

Скопируйте сгенерированную ссылку внизу, перейдите по ней и пригласите бота на свой сервер.

Шаг 2: Установка необходимых программ и библиотек
Вам понадобится установленный на компьютере Python (версии 3.8 или новее).

Также для работы со звуком обязательно нужен FFmpeg:

Windows: Скачайте сборку FFmpeg, распакуйте и добавьте путь к папке bin в системные переменные среды (PATH).

macOS: Откройте терминал и введите brew install ffmpeg (нужен Homebrew).

Linux: Откройте терминал и введите sudo apt install ffmpeg.

Затем откройте терминал (командную строку) и установите нужные библиотеки Python:

Bash
pip install discord.py yt-dlp pynacl
Шаг 3: Написание кода
Создайте файл bot.py, откройте его в любом текстовом редакторе (например, VS Code) и вставьте следующий код:

Python
import discord
from discord.ext import commands
import yt_dlp
import asyncio

# Настройка намерений бота
intents = discord.Intents.default()
intents.message_content = True

# Префикс для команд (например, !play)
bot = commands.Bot(command_prefix='!', intents=intents)

# Настройки для yt-dlp (получаем лучшее доступное аудио)
ydl_opts = {
    'format': 'bestaudio/best',
    'noplaylist': True,
    'quiet': True,
}

# Настройки для FFmpeg (чтобы стрим не прерывался)
ffmpeg_options = {
    'before_options': '-reconnect 1 -reconnect_streamed 1 -reconnect_delay_max 5',
    'options': '-vn'
}

@bot.event
async def on_ready():
    print(f'Бот {bot.user} успешно запущен и готов к работе!')

@bot.command(name='play', help='Воспроизводит музыку с YouTube по ссылке')
async def play(ctx, url: str):
    # Проверка, находится ли пользователь в голосовом канале
    if not ctx.message.author.voice:
        await ctx.send("Вы должны находиться в голосовом канале, чтобы включить музыку!")
        return

    channel = ctx.message.author.voice.channel

    # Подключение к голосовому каналу
    voice_client = discord.utils.get(bot.voice_clients, guild=ctx.guild)
    if not voice_client:
        voice_client = await channel.connect()
    elif voice_client.channel != channel:
        await voice_client.move_to(channel)

    await ctx.send(f"Ищу трек... 🎵")

    # Получение прямой ссылки на аудиопоток
    try:
        with yt_dlp.YoutubeDL(ydl_opts) as ydl:
            info = ydl.extract_info(url, download=False)
            stream_url = info['url']
            title = info.get('title', 'Неизвестный трек')

        # Остановка текущего трека, если он играет
        if voice_client.is_playing():
            voice_client.stop()

        # Воспроизведение
        source = discord.FFmpegPCMAudio(stream_url, **ffmpeg_options)
        voice_client.play(source)
        await ctx.send(f"Сейчас играет: **{title}** 🎶")

    except Exception as e:
        await ctx.send("Произошла ошибка при загрузке видео. Возможно, это прямая трансляция или видео недоступно.")
        print(f"Ошибка: {e}")

@bot.command(name='stop', help='Останавливает музыку и выгоняет бота')
async def stop(ctx):
    voice_client = discord.utils.get(bot.voice_clients, guild=ctx.guild)
    if voice_client and voice_client.is_connected():
        await voice_client.disconnect()
        await ctx.send("Бот отключен от канала. Пока! 👋")
    else:
        await ctx.send("Бот сейчас не в голосовом канале.")

# ЗАМЕНИТЕ НА ВАШ ТОКЕН
bot.run('СЮДА_ВСТАВЬТЕ_ВАШ_ТОКЕН')
Шаг 4: Запуск
В терминале перейдите в папку, где лежит ваш файл bot.py, и запустите его:

Bash
python bot.py
Зайдите в голосовой канал на вашем сервере и напишите в текстовый чат: !play <ссылка на видео YouTube>. Бот должен зайти к вам и начать проигрывать музыку.

Установка FFmpeg (особенно на Windows) часто вызывает сложности у новичков. Удалось ли вам разобраться с его установкой, или нужна помощь с прописыванием системных переменных PATH?
